# Portafolio de Luis Sauceda

Portafolio personal desarrollado con Angular para mostrar mi perfil como Full-Stack Developer, proyectos destacados, experiencia técnica y canales de contacto para reclutadores y clientes.

## Demo

- Sitio en vivo: `https://luissauceda93.github.io/portafolio/`

## Capturas

- Desktop: agrega una captura del layout principal
- Mobile: agrega una captura del menú lateral y la vista móvil

## Características

- Hero principal con propuesta de valor
- Navegación fija en desktop
- Menú lateral en móvil
- Sección de tecnologías por áreas
- Proyectos destacados con enfoque en impacto
- Resumen de experiencia profesional
- Llamados a la acción para contacto y CV

## Tecnologías

- Angular
- TypeScript
- SCSS
- HTML5
- GitHub Pages

## Requisitos

- Node.js
- npm

## Instalación local

```bash
npm install
npm start
```

Luego abre `http://localhost:4200/`.

## Scripts disponibles

```bash
npm start
npm run build
npm test
```

## Estructura general

- `src/app/app.component.ts`: datos y estado de la vista
- `src/app/app.component.html`: estructura principal del portafolio
- `src/app/app.component.scss`: estilos de la landing
- `src/styles.scss`: estilos globales
- `src/index.html`: metadatos y título de la página
